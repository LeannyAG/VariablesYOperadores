# VariablesYOperadores

# descripción del Proyecto
Este programa en Java demuestra el uso de variables, operadores aritméticos, operadores lógicos y estructuras de control. 
El programa solicita datos al usuario y realiza diferentes validaciones y clasificaciones.todo en uno solo 


 Respuestas Teóricas

## ¿Para qué se utilizan los operadores lógicos en programación?

Los operadores lógicos se utilizan para evaluar múltiples condiciones al mismo tiempo. 
Permiten combinar expresiones booleanas usando AND (&&), OR (||) y NOT (!), 
lo que ayuda a tomar decisiones dentro del programa.


# ¿Por qué es importante declarar correctamente el tipo de dato de una variable?

Es importante porque cada tipo de dato ocupa diferente espacio en memoria 
y permite realizar ciertas operaciones específicas. 
Si se declara incorrectamente, el programa puede generar errores 
o comportamientos inesperados.



 Justificación Técnica

Clasificación de edad
Se utilizó la estructura de control `if - else if - else` 
porque permite evaluar múltiples rangos de edad 
y ejecutar una acción específica según la condición que se cumpla.

Día de la semana
Se utilizó la estructura `switch` 
porque permite evaluar múltiples opciones de manera más ordenada 
cuando se trabaja con valores específicos como números o días.

 Verificación de acceso
Se utilizaron operadores `if` equals 
para validar que se cumplan múltiples condiciones al mismo tiempo, 
como edad mínima o credenciales correctas.
